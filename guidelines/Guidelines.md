# 무무목장 홈페이지 — AI Cowork 가이드라인

> 이 문서는 Claude Cowork(또는 다른 AI 도구)가 무무목장 홈페이지 코드를 수정·개선할 때 반드시 따라야 하는 지침입니다.

---

## 1. 절대 원칙

### 1-1. 콘텐츠 정확성
- **무무목장은 유제품 목장**입니다. 여행사, 투어 회사가 아닙니다.
- 모든 텍스트는 "강원도 삼척 예수원 목장", "저지종 젖소", "A2 우유", "수제 치즈/요거트"와 관련된 내용이어야 합니다.
- "여행", "투어", "탐험", "관광", "보르도", "Traravellio" 등 여행 관련 문구가 있으면 즉시 삭제하고 목장 관련 내용으로 교체하세요.
- 의미가 통하지 않는 깨진 한국어(예: "숲허락도록 이루너는") 문장은 절대 남기지 마세요. 발견 즉시 자연스러운 한국어로 다시 작성하세요.

### 1-2. 디자인 스타일
- 현재 홈페이지는 **풀스크린 배경 사진, 플로팅 이미지, 다크 섹션**을 활용한 모던하고 시각적으로 임팩트 있는 스타일을 채택하고 있습니다.
- 이 비주얼 스타일은 의도적으로 선택된 것이므로 유지합니다.
- `src/imports/pasted_text/moomoo-ranch-design-guide.md`는 색상 토큰, 타이포, 기본 컴포넌트 스타일의 참고 자료로 활용하되, 레이아웃/섹션 구조는 현재 구현을 기준으로 합니다.

## 2. 비주얼 스타일 원칙 (현재 구현 기준)

현재 홈페이지는 아래 비주얼 패턴을 사용합니다. 새 섹션이나 페이지를 만들 때 이 패턴을 따르세요:

### 히어로 섹션 패턴
- **풀스크린 배경 이미지** + 어둡게 오버레이 (`brightness(0.6~0.7)` + gradient)
- 플로팅 이미지 (비스듬히 기울어진 rounded-3xl 카드, `hidden lg:block`)
- 중앙 정렬 대형 타이틀 (흰색, `clamp()` 반응형 사이즈)
- 통계/하이라이트 바 (아이콘 + 텍스트)

### 콘텐츠 섹션 패턴
- **2열 레이아웃**: 이미지 | 텍스트 (또는 반대), `lg:grid-cols-2`, 앞뒤 교차
- **카드 그리드**: `md:grid-cols-2 lg:grid-cols-4`, 흰 배경 카드 + 미세 그림자
- **다크 섹션**: `#1a1a1a` 배경, 배경 이미지 30% opacity, 흰 텍스트
- **갤러리**: 가로 스크롤 카드 리스트, 세로형(320×420px) 이미지 카드

### CTA 섹션 패턴
- 풀폭 배경 이미지 (`brightness(0.4~0.5)`)
- 중앙 정렬 대형 텍스트 + pill 버튼 (흰 배경)

### 공통 UI 요소
- 섹션 라벨: pill 형태의 태그 (`rounded-full`, 반투명 배경 + 보더)
- 모든 카드/이미지: `rounded-3xl`
- 모든 CTA 버튼: `rounded-full` (pill), `hover:scale-105`
- 섹션 간 충분한 패딩: `py-24 md:py-32`
- Framer Motion 애니메이션: `whileInView`, `initial/animate`

## 3. 색상 시스템

CSS 변수(`theme.css`)에 이미 정의된 토큰을 사용하세요:

| 용도 | 변수 | HEX |
|------|------|-----|
| 전체 배경 | `var(--cream)` | #FFF8E1 |
| 주 액센트 | `var(--dusty-rose)` | #D4A0A0 |
| CTA 버튼 | `var(--pale-orange)` | #FFCC80 |
| 강조/hover | `var(--golden)` | #FFD54F |
| 자연/목장 섹션 | `var(--sage)` | #A5D6A7 |
| 링크/정보 섹션 | `var(--soft-blue)` | #90CAF9 |
| 본문 텍스트 | `var(--text-main)` | #4E342E |
| 보조 텍스트 | `var(--text-light)` | #8D6E63 |
| 카카오 CTA | `var(--kakao-yellow)` | #FEE500 |

규칙:
- 배경에 색상을 쓸 때는 10~30% opacity로 (예: `rgba(165, 214, 167, 0.15)`)
- 본문 텍스트는 항상 `--text-main`, 보조는 `--text-light`
- 밝은 섹션 배경: `var(--cream)` 또는 `rgba(255,255,255,0.5)`
- 다크 섹션 배경: `#1a1a1a`
- 헤더: 반투명 흰색 (`rgba(255,255,255,0.85)`) + `backdrop-blur-md`
- 푸터: `#1a1a1a` 다크 배경

## 5. 홈페이지 섹션 구조 (현재 구현 기준)

1. **히어로**: 풀스크린 배경 이미지 + 플로팅 사진 카드 4장 + 대형 타이틀 + CTA 버튼 + 통계 바
2. **목장 소개**: 2열 레이아웃 (텍스트 | 이미지 그리드), sage 계열 라벨
3. **특징 카드 그리드**: 4열 이미지+텍스트 카드 (목장, 방목, 우유, 치즈)
4. **다크 섹션 — 저지종 소개**: 어두운 배경 + 배경 이미지 3단 + 세로형 피처 카드
5. **갤러리**: 가로 스크롤 사진 갤러리 + 해시태그 안내
6. **CTA**: 풀폭 배경 이미지 + "제품 구경하기" CTA
7. **푸터 이모지 바**: 제품/키워드 나열 (🥛 A2 우유, 🧀 수제 치즈, 🐄 저지종 젖소 등)

## 6. 컴포넌트 스타일 규칙

### 카드
- 배경: 흰색
- 모서리: `rounded-3xl` (24px)
- 보더: `1px solid rgba(141, 110, 99, 0.08)` (미세한 보더)
- 그림자: `0 4px 20px rgba(0,0,0,0.06)`, hover 시 `scale(1.02)`

### 버튼
- CTA Primary: 흰 배경, `text-main` 텍스트, pill(`rounded-full`), `boxShadow`, hover → `scale(1.05)`
- CTA Sage: `var(--sage)` 배경, 흰 텍스트, pill
- 카카오 CTA: `#FEE500` 배경, `text-main` 텍스트, pill
- 최소 패딩: `px-8 py-4` (큰 CTA는 `px-10 py-5`)

### 치즈 아바타
- `CheeseAvatar` 컴포넌트 활용 가능
- 사이즈: SM(32px) / MD(48px) / LG(64px) / XL(96px)

### 섹션 라벨 (pill 태그)
```jsx
<div className="inline-flex items-center gap-2 mb-6 px-4 py-2 rounded-full" style={{
  backgroundColor: 'rgba(색상, 0.15)',
  border: '1px solid rgba(색상, 0.3)'
}}>
```

## 7. 코드 품질 규칙

### 인라인 스타일
- 현재 코드는 `style={{}}` 인라인 스타일을 많이 사용하고 있습니다. 일관성을 위해 기존 패턴을 따르되, 새 코드에서는 가능한 Tailwind 클래스를 우선 활용하세요.
- CSS 변수 참조가 필요한 경우(color, fontFamily 등)는 인라인 스타일 사용 가능.

### 이미지 처리
- Unsplash URL에 적절한 `w=` 파라미터를 사용하세요 (히어로: 1920, 카드: 800, 썸네일: 400)
- `loading="lazy"` 속성을 스크롤 아래(below-the-fold) 이미지에 추가하세요
- `alt` 텍스트를 구체적으로 작성하세요 (예: "무무목장 저지종 젖소가 초원에서 풀을 뜯는 모습")

### 접근성
- 모든 인터랙티브 요소에 `focus-visible` 스타일 추가
- 색상 대비 최소 4.5:1 유지 (WCAG AA)
- 시맨틱 HTML 사용: `<section>`, `<nav>`, `<article>`, `<main>`
- 아이콘 전용 버튼에 `aria-label` 추가

### 애니메이션
- Framer Motion (`motion/react`) 사용
- 진입 애니메이션: `initial={{ opacity: 0, y: 30 }}` → `whileInView={{ opacity: 1, y: 0 }}`
- `viewport={{ once: true }}` 필수 (한 번만 실행)
- duration: 0.6~0.8초, delay로 순차 등장

## 9. 자주 하는 실수 방지 체크리스트

코드를 수정한 후 아래 항목을 점검하세요:

- [ ] 여행/투어 관련 문구가 남아있지 않은가?
- [ ] 깨진 한국어(의미 불명 문장)가 없는가?
- [ ] 색상이 theme.css의 CSS 변수를 사용하는가?
- [ ] 버튼 터치영역이 48px 이상인가?
- [ ] 본문 글씨가 16px 이상, 줄간격 1.6 이상인가?
- [ ] 이미지에 구체적인 alt 텍스트가 있는가?
- [ ] 스크롤 아래 이미지에 loading="lazy"가 있는가?
- [ ] 모바일 반응형이 제대로 동작하는가?
- [ ] Framer Motion에 viewport={{ once: true }}가 있는가?
